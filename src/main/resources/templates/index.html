暨大论坛一个Java开发的web项目，主要功能有登录、注册、浏览话题、发表话题、话题分类浏览、评论、个人用户的积分、点赞（置为有用评论）、站内信等等。 它是一个校内讨论交流的平台，师生可以畅所欲言。

Technologies

Java 8 Standard Edition、 Spring MVC、 Spring Data、 Spring Security、 Spring Boot、 MyBatis、 HTML （Themyleaf）、 CSS （Materialize）

Software tools

Eclipse、 Maven、 MySQLWorkbench [MySQL]、PostMan
